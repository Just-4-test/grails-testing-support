
=== The Basics

Use the `grails.testing.web.controllers.ControllerUnitTest` trait to unit
test controllers.

----
include::{demoAppDir}/src/test/groovy/demo/DemoControllerSpec.groovy[tags=basic_declaration,indent=0]
----

Adding the `ControllerUnitTest` trait to a test causes a property named
`controller` to be added to the test.  This property will automatically be
 initialized with a instance of the controller under test.
